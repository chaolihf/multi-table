## ADDED Requirements

### Requirement: 数据筛选
系统 SHALL 支持按条件筛选显示特定的行。

#### Scenario: 启用筛选
- **WHEN** 用户点击"筛选"按钮
- **THEN** 每列标题显示筛选下拉箭头

#### Scenario: 文本筛选
- **WHEN** 用户点击文本列的筛选箭头
- **THEN** 显示筛选选项：包含、等于、开头是、结尾是
- **AND** 输入筛选值后，只显示匹配的行

#### Scenario: 数字筛选
- **WHEN** 用户点击数字列的筛选箭头
- **THEN** 显示筛选选项：大于、小于、等于、介于
- **AND** 输入筛选值后，只显示匹配的行

#### Scenario: 清除筛选
- **WHEN** 用户点击"清除筛选"按钮
- **THEN** 显示所有行，恢复原始数据视图

#### Scenario: 多列筛选
- **WHEN** 用户在多列上设置筛选条件
- **THEN** 只显示同时满足所有条件的行
