## ADDED Requirements

### Requirement: 数据图表
系统 SHALL 支持根据选中的数据区域生成可视化图表。

#### Scenario: 创建柱状图
- **WHEN** 用户选中数据区域并点击"柱状图"按钮
- **THEN** 在新窗口中显示柱状图
- **AND** 第一列作为 X 轴标签，其他列作为数据系列

#### Scenario: 创建折线图
- **WHEN** 用户选中数据区域并点击"折线图"按钮
- **THEN** 在新窗口中显示折线图
- **AND** 第一列作为 X 轴标签，其他列作为数据系列

#### Scenario: 创建饼图
- **WHEN** 用户选中单列数据并点击"饼图"按钮
- **THEN** 在新窗口中显示饼图
- **AND** 第一列作为标签，第二列作为数值

#### Scenario: 更新图表
- **WHEN** 源数据发生变化
- **THEN** 图表显示刷新按钮，点击后更新数据

#### Scenario: 导出图表
- **WHEN** 用户点击"导出图片"按钮
- **THEN** 下载图表为 PNG 图片文件
