## MODIFIED Requirements

### Requirement: 单元格选择
系统 SHALL 支持选择单个单元格、连续区域、不连续区域、整行、整列。

#### Scenario: 选择单个单元格
- **WHEN** 用户点击单元格 A1
- **THEN** A1 被高亮显示为选中状态
- **AND** 后续键盘输入可进入编辑模式

#### Scenario: 选择连续区域
- **WHEN** 用户从 A1 拖动到 C3
- **THEN** A1:C3 区域被高亮选中

#### Scenario: 选择不连续区域
- **WHEN** 用户按住 Ctrl 键依次点击 A1、C3、E5
- **THEN** A1、C3、E5 三个单元格同时被选中

#### Scenario: 选择整行
- **WHEN** 用户点击行标题 "5"
- **THEN** 第 5 行所有单元格被选中
- **AND** 选中区域为第 5 行的所有列

#### Scenario: 选择整列
- **WHEN** 用户点击列标题 "C"
- **THEN** C 列所有单元格被选中
- **AND** 选中区域为 C 列的所有行

### Requirement: 双击编辑
系统 SHALL 支持双击单元格进入编辑模式。

#### Scenario: 双击进入编辑
- **WHEN** 用户双击单元格 A1
- **THEN** A1 进入编辑模式，显示输入框
- **AND** 输入框获得焦点

#### Scenario: 双击后取消编辑
- **WHEN** 用户双击进入编辑模式后按 Escape
- **THEN** 编辑取消，单元格恢复原始值

### Requirement: 键盘输入编辑
系统 SHALL 支持在单元格选中后直接键盘输入进入编辑模式。

#### Scenario: 字母键输入
- **WHEN** 用户选中 A1 并按下字母键 "A"
- **THEN** A1 进入编辑模式，显示输入 "A"
- **AND** 光标位于字符后

#### Scenario: 数字键输入
- **WHEN** 用户选中 A1 并按下数字键 "1"
- **THEN** A1 进入编辑模式，显示输入 "1"

#### Scenario: F2 进入编辑
- **WHEN** 用户选中 A1 并按下 F2 键
- **THEN** A1 进入编辑模式，光标位于末尾

### Requirement: 右键菜单操作
系统 SHALL 在用户右键点击时显示上下文菜单，操作以选中区域为基准。

#### Scenario: 显示右键菜单
- **WHEN** 用户右键点击选中的单元格
- **THEN** 显示菜单：复制、粘贴、剪切、删除、清除内容、插入行、插入列
- **AND** 菜单位置靠近鼠标指针

#### Scenario: 复制选中区域
- **WHEN** 用户选中 A1:B2 并右键选择"复制"
- **THEN** A1:B2 区域内容被复制到剪贴板
- **AND** 不是鼠标位置下的单元格

#### Scenario: 粘贴到选中区域
- **WHEN** 用户选中 A1 并右键选择"粘贴"
- **THEN** 剪贴板内容粘贴到以 A1 为起点的区域
- **AND** 不是鼠标位置下的单元格

#### Scenario: 插入行
- **WHEN** 用户选中第 5 行的单元格并右键选择"插入行"
- **THEN** 在第 5 行位置插入新行
- **AND** 不是鼠标位置下的行

#### Scenario: 插入列
- **WHEN** 用户选中 C 列的单元格并右键选择"插入列"
- **THEN** 在 C 列位置插入新列
- **AND** 不是鼠标位置下的列
